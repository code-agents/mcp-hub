# <Icon icon="laptop-code" iconType="solid" color="#3B82F6" /> MCP開発ホスト

MCP開発ホストは、AI駆動アプリケーションの構築、テスト、および対話のための特殊な環境を提供します。このセクションでは、MCP開発を強化する様々なホストを紹介します。

## <Icon icon="star" iconType="solid" /> 人気のMCPホスト

- <Icon icon="keyboard" iconType="solid" /> **Cursor**: インラインAIアシスタンスを備えたMCPアプリケーション開発に最適化されたAI駆動コードエディタ
- <Icon icon="desktop" iconType="solid" /> **Claude Desktop**: Claudeモデルと直接対話するためのスタンドアロンアプリケーション
- <Icon icon="terminal" iconType="solid" /> **Cline**: Claude駆動アプリケーションの構築とテストのためのコマンドラインインターフェース
- <Icon icon="puzzle-piece" iconType="solid" /> **VSCode拡張機能**: Visual Studio Codeエディタ向けのAI統合
- <Icon icon="wind" iconType="solid" /> **Windsurf**: MCPコンポーネントの開発とテストのためのブラウザベースインターフェース

適切な開発ホストは、AI開発に合わせた特殊なツール、インターフェース、ワークフローを提供することで、MCPアプリケーション構築時の生産性を大幅に向上させることができます。
