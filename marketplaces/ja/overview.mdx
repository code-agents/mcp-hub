MCPマーケットプレイスは、MCPコンポーネントやアプリケーションの発見、配布、収益化のためのプラットフォームを提供します。これらの特化した取引所は、MCPエコシステムの成長を促進します。

## <Icon icon="star" iconType="solid" /> MCPレジストリとマーケットプレイス

<Note>
現在、最適なMCPマーケットプレイスを選定するためにこのリストの精査を進めています。
</Note>

- [**modelcontextprotocol/servers**](https://github.com/modelcontextprotocol/servers): Model Context Protocol (MCP)のリファレンス実装のリポジトリ
- [**MCP Market**](https://mcpmarket.com): お気に入りのツールにAIを接続するためのMCPサーバーの厳選コレクション
- [**MCP.so**](https://mcp.so): 優れたMCPサーバーとクライアントを見つける
- [**Smithery**](https://smithery.ai): Model Context Protocolサーバーを通じて4,274の機能であなたのエージェントを拡張
- [**Pulse MCP**](https://www.pulsemcp.com): MCPのユースケース、サーバー、クライアント、ニュースを閲覧・発見
- [**Glama MCP Server**](https://glama.ai/mcp/servers): AI機能を拡張する本番対応および実験的なMCPサーバー
- [**Cursor MCP Registry**](https://cursor.directory/mcp): MCPを閲覧したり、月間25万人以上のアクティブな開発者にリーチするためにMCPを投稿
- [**Awesome MCP servers**](https://github.com/punkpeye/awesome-mcp-servers): Model Context Protocol (MCP)サーバーの厳選リスト
- [**MCP Servers**](https://mcpservers.org): Model Context Protocolのサーバーコレクション
- [**OpenTools**](https://opentools.com/registry): 160以上のMCPサーバーにわたる400以上のツールの機能を文書化したレジストリ
- [**MCP Composio**](https://mcp.composio.dev): 組み込み認証を備えた100以上の管理されたMCPサーバーに即時接続
- [**MCP-Get**](https://mcp-get.com): プロトコルサーバーを簡単にインストールするのに役立つ
- [**Gumloop**](https://www.gumloop.com/mcp): MCPサーバーの網羅的なリスト
- [**API Tracker**](https://apitracker.com): 最高のAPIと開発者リソースを発見
- [**Apify**](https://apify.com): Webスクレイピングのためのフルスタックプラットフォーム
- [**Cline.bot**](https://cline.bot/mcp-marketplace): Cline.bot用のMCPサーバー
- [**Dexter MCP**](https://www.dextermcp.net): さまざまなタスク用の特殊なMCPサーバーであなたのAIを強化
- [**Fleur**](https://www.fleurmcp.com): Claudeのアプリストア
- [**Make MCP**](https://developers.make.com/mcp-server): Makeオートメーションプラットフォームで1000以上のアプリにAIモデルを接続
- [**MCP Repository**](https://mcprepository.com): Model Context Protocolの究極のリソース
- [**MCP Resolver**](https://mcpresolver.com): AIアプリケーションに適切なMCPサーバーを見つけて接続
- [**MCP Run**](https://www.mcp.run): 組織全体の垂直AIのためのワンプラットフォーム
- [**MCP Store**](http://mcpstore.co): AIアプリケーション用のさまざまなMCPサーバーを発見して使用
- [**Pipedream MCP**](https://mcp.pipedream.com): 統合のためのAI開発者ツールキット
- [**SuperMachine**](https://supermachine.ai): 開発者やビジネス向けのAIパワードツールとMCPサーバー
- [**Zapier MCP**](https://zapier.com/mcp): Zapier MCPであなたのAIを任意のアプリに接続

これらの専用MCPマーケットプレイスは、開発者が既製のコンポーネントを発見し、自分の作品を共有し、ゼロから始めるのではなく既存のソリューションに基づいて開発を加速するのに役立ちます。
