# MCPVerifiedの価値観

MCPVerifiedでは、MCPアプリケーションの成功につながる技術や実践について明確な意見を持っています。これらの価値観は私たちの推奨事項やコンテンツを導くものです。

## 私たちの核となる価値観

- **TypeScriptファースト**: TypeScriptは本番環境向けAIアプリケーションの言語であると考えています。型安全性と開発者体験の適切なバランスを提供します。

- **パフォーマンスは重要**: MCPアプリケーションは後付けではなく、初めからパフォーマンスを意識して構築されるべきです。

- **開発者体験**: ツールやフレームワークは本番品質を維持しながら、優れたDXを優先すべきです。

- **モジュラーアーキテクチャ**: コンポーネントはモジュール化され、再利用可能で、適切に文書化されており、エコシステムの成長を促進すべきです。

- **デフォルトでサーバーレス**: やむを得ない理由がない限り、サーバーレスアーキテクチャは大部分のMCPアプリケーションに最適なスケーリングとメンテナンスプロファイルを提供します。

- **セキュリティ優先**: 推奨するサービス、パッケージ、サーバーについて、セキュリティの脆弱性とベストプラクティスを慎重に検証しています。

- **独自の見解によるキュレーション**: 最も効果的なものについて強い意見を持つことを恐れません。私たちの推奨事項は実世界の経験に基づいて慎重にキュレーションされています。

- **公式パッケージを優先**: 利用可能な場合は、より良いサポートと信頼性のために、サードパーティの代替品よりも公式パッケージと実装を推奨します。

これらの意見は、MCPアプリケーションを構築した私たちの集合的な経験に基づいており、このエコシステムで成功するための最良の道筋であると考えています。
