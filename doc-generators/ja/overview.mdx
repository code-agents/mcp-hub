# <Icon icon="book" iconType="solid" color="#3B82F6" /> ドキュメント生成ツール

ドキュメントはMCPプロジェクトの重要な部分です。このセクションでは、MCP アプリケーションに特化したサポートを提供するドキュメント生成ツールを紹介します。

## MCP特化型生成ツールの重要性

MCP互換ドキュメント生成ツールは、従来の検索拡張生成（RAG）アプローチと比較して、特にコンテキスト長の効率性に関して大きな利点を提供します。

従来のRAGでは、AIのコンテキストウィンドウに大量のドキュメントを読み込むことが多く、その多くは特定のタスクに関連しない可能性があります。これは貴重なコンテキスト空間を消費し、AIの集中力を低下させる可能性があります。

対照的に、MCPではAIエージェントやIDEが必要に応じてドキュメントサーバーから特定の情報を検索できます。潜在的に関連性のないデータを事前に読み込む代わりに、エージェントは必要なときに必要な詳細（APIの使用法、例、変更ログなど）だけを取得します。この的を絞った検索により、コンテキストウィンドウがスリムで焦点が合った状態を保ち、AIアシスタントによるタスクの完了がより速く効率的になります。このアプローチは、Mastra MCPドキュメントサーバーのようなシステムで見られる利点を反映しており、エージェントがコンテキストを過負荷にすることなく完全な知識ベースにアクセスできるようにします（[出典：Mastra MCPドキュメントサーバーの紹介](https://mastra.ai/blog/introducing-mastra-mcp)）。

## <Icon icon="star" iconType="solid" /> MCP互換ドキュメント生成ツール

- <Icon icon="leaf" iconType="solid" /> **Mintlify**: このサイトで使用されているモダンなドキュメントプラットフォームで、AI関連コンテンツやインタラクティブな例のサポートが組み込まれています
- <Icon icon="microphone-alt" iconType="solid" /> **Speakeasy**: AIエンドポイントや機能を文書化するための特別な機能を備えたAPIドキュメント生成ツール
- <Icon icon="shield-alt" iconType="solid" /> **Stainless**: MCPサービス向けの堅牢なAPIリファレンスとSDKドキュメントに焦点を当てたドキュメントツール

これらのドキュメントツールは、APIエンドポイント、モデル機能、インタラクションパターンなど、MCPアプリケーションの独自の側面を説明、実演、文書化するための特殊な機能を提供します。
