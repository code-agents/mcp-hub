## How to Use

To use this component in your MDX files:

1. Import the component:
```jsx
import { GitHubPackageCards } from '/snippets/GitHubPackageCards.mdx';
```

2. Use the component with an array of GitHub repositories:
```jsx
<GitHubPackageCards 
  packages={[
    {
      owner: "owner-name",      // GitHub owner/organization
      repo: "repo-name",        // GitHub repository name
      title: "Display Title",   // Optional: Custom title (defaults to npmPackage or repo)
      npmPackage: "package-name" // Optional: npm package name (defaults to repo name)
    }
    // Add more packages as needed
  ]}
/>
```
