## <Icon icon="star" iconType="solid" /> 推奨 MCP サーバー

- <Icon icon="github" iconType="brands" /> **GitHub 公式 MCP**: 包括的なドキュメントとコミュニティサポートを備えたリファレンス実装
- <Icon icon="aws" iconType="brands" /> **AWS MCP**: AWS Lambda と API Gateway に最適化されたサーバーレス実装

- <Icon icon="microsoft" iconType="brands" /> **Microsoft Playwright MCP**: Web 自動化とテストシナリオに特化した実装
